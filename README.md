# Java-Collections-Demo-ArrayList-Vector-Iterator-HashMap



Ce petit projet Java est une démonstration pratique de l’utilisation des **collections Java** à travers un exemple simple de gestion de membres (`Membre`).

Il illustre l’usage de :
- `ArrayList`
- `Vector`
- `Iterator`
- `HashMap`
- Boucles `for`, `for-each`
- Méthodes fondamentales des collections (`add`, `get`, `set`, `remove`, `size`)
